Sweet Delight

概要

この作品は、オンライン講座「デイトラ」での課題として制作したコーディングサンプルです。
ひと目で分かりやすいスウィーツ店のデモサイトを構築しました。

使用技術

HTML CSS jQuery

制作期間

2週間

学んだこと

  • pictureタグで画像レスポンシブの切り替え
  • Emmetとユーザースニペットで効率化
  • jQueryを使用し、動きを出す

工夫した点

  • hover時に指定した時間でカーソルが当たるようにした
  • Menuのケーキ画像をクリックするとモーダルになり、その際後ろでスクロールできないようにしました。

ソースコード

ソースコードはGitHubのプライベートリポジトリにて管理しています。必要な場合はアクセス権を提供いたしますので、お問い合わせください。

反省点

  • Emmetを使えるところで使えてないときがあった。時短のためにも積極的に使用していく。
