目覚ましアプリサイト OHA!

概要

この作品は、オンライン講座「デイトラ」での課題として制作したレスポンシブデザインのコーディングサンプルです。
ユーザーのデバイスに応じてレイアウトが変化するサイトを構築しました。

使用技術

HTML CSS Sass

JavaScript jQuery Figma

制作期間

1ヶ月

学んだこと

  • メディアクエリを使用したレスポンシブデザインの基礎
  • Figmaからの値の取得や素材の書き出し
  • 徹底したピクセルパーフェクト

工夫した点

  • アニメーションを使って、ユーザーインタラクションを向上
  • ピクセルパーフェクトのために、各セクションを作り終えるたびにその都度チェックし、ズレがないか確認

プレビュー

スクリーンショットギャラリー ビデオデモ PDFドキュメント

ソースコード

ソースコードはGitHubのプライベートリポジトリにて管理しています。必要な場合はアクセス権を提供いたしますので、お問い合わせください。

反省点

  • レスポンシブデザインの基礎を学ぶにあたり、メディアクエリの使い方や、デバイスごとのレイアウトの違いをもっと理解する必要があると感じました。今後は、実際のプロジェクトでの経験を積み、より高度なレスポンシブデザインを実現できるようになりたいと考えています。
  • また、制作時間がかかりすぎてしまったため、今後は効率的なコーディング方法を模索し、制作時間の短縮を図りたいです。
